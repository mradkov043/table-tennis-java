TableTennis – Zuerst muss der Spieler eine Anzahl von Bällen in das Textfeld (int) eingeben 
und einen der fünf Schwierigkeitsgrade wählen. Um das Spiel zu starten, muss der Spieler die Taste "Play!" klicken.

Es gibt drei Funktionstasten : mit "A" und "D" kann man den Schläger bewegen und mit "ESC" kann man das Spiel früher beenden.
Das kann man auch mit einem Klick auf den Button "Give Up"  machen.
 
Wenn das Spiel endet, hat der Spieler 2 Optionen : 
"Restart" clicken um noch einmal zu spielen oder "Exit" um das Programm zu schließen.
 
Die Spiellogik wird mithilfe einer Timeline implementiert, die die Positionen 
der Bälle aktualisiert und Kollisionen mit dem Schläger und den Bildschirmgrenzen überprüft.
